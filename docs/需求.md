# 背景
我希望在这个项目中制作一个ClaudeCode的使用教程，包括CLI以及VSCode和Jetbrains的插件，提供一些基础和进阶使用技巧

# 实现方式
我希望将教程以网页形式部署在我们的内网（没有互联网）服务器上，以便提供美观页面，也确保我随时能够更新维护
- 技术栈请你来决定，不过内网服务器上有nodejs
- 提供一个简单好记的内网访问链接，比如claude.com

# 教程内容
教程不仅面向开发人员（主要），有时也面向一般没有编程经验的人员，内容我希望从基础到进阶循序渐进
包括但不限于：
1. ClaudeCode的安装方式（这个留白，我来写）
2. ClaudeCode基本介绍，
3. 模式切换，不同模式之间的区别，使用场景和示例
4. Memory的用法，包括用户级和项目级，比如在用户级memory中说明语言规范（使用中文回答问题，中文注释，指明运行环境windows，从而避免执行linux格式命令浪费上下文和时间等） 
5. /init 的用法
6. /compact 的用法
7. 使用! 切换命令行
8. /permissions 权限管理，比如禁止联网查询的功能，因为没有互联网
9. 自定义命令，.claude/commands 自定义命令参数
10. SubAgent的用法
11. resume
12. 与版本控制git相结合
13. 需求先行，建立docs文件夹，将步骤分解细化
14. claude --dangerously-skip-permissions

不限于以上内容，你可以进行补充和完善，也不限于上面的出现顺序
